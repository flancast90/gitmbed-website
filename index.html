As a Chrome Extension, Gitmbed checks every page for a gitmbed element. If one is found, it renders the the embed using options specified in the source of the gitmbed element. Gitmbed is served through an anchor element (link), with the additional arguments appended to the href.

Manual-Gitmbed

  1. The bare-minimum for gitmbed content is that the page contains an anchor element () with the HTML href attribute. Inside this href, you must append the arguments embed src, embed width, embed height
For example, a simple interpretation of this (acceptable in GitHub markdown) would look like this:
<a href="https://github.com/flancast90/gitmbed#gitmbed=https://tiddlywiki.com,100%,500"><img width="100%" src="https://gitmbed.finnsoftware.net/assets/img/button_gitmbed.png"/></a>
