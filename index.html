Download

Chrome Web Store

  1. Open the Chrome Web Store.
  2. Search for "gitmbed" in the search bar, and select the gitmbed extension.
  3. Click Add to Chrome.
  4. That's It, you've successfully downloaded gitmbed! Enjoy!


Load Unpacked (Developer Option)

Loading an extension via this method allows further control over the extension, particularly if you will be adding-onto or developing this extension further. Please note that this method can also be dangerous to your computer. See https://courses.csail.mit.edu/6.857/2019/project/6-Li-Rosales-Yang.pdf for more information.

  1. Visit chrome://extensions (via omnibox or menu -> Tools -> Extensions).
  2. Enable Developer mode by ticking the checkbox in the upper-right corner.
  3. Click on the "Load unpacked extension..." button.
  4. Select the directory containing your unpacked extension.
These instructions are for the New Microsoft Edge, and do not support the Legacy Editions.

Features

Easy-to-Use

Gitmbed is easy to use! All you have to do is download the extension, create a new gitmbed, and then add it to your post, profile, etc.


Supports GitHub, Instagram, and Other Websites/Apps

Gitmbed has a simple-setup via Instagram and GitHub. While using the extension, you can easily create new gitmbeds, which will be visible to anyone else with the extension!


Guaranteed Reliability

With gitmbed, you can expect top-notch support, and guaranteed reliability. Because of the simplistic yet powerful approach gitmbed takes to its content, debugging is easy, and support is even-easier!
